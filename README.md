# Calculadora Grêmio

Calculadora web com tema do Grêmio Foot-Ball Porto Alegrense, construída ponta a ponta com o AIOX Story Development Cycle.

## ✨ Demonstração
A calculadora está hospedada via GitHub Pages. Acesse pelo link:
👉 **[https://henriquefreitas-stark.github.io/calculadora-gremio/](https://henriquefreitas-stark.github.io/calculadora-gremio/)**

Como alternativa, você pode abrir localmente:
`
index.html
`

## 🏗️ Tecnologias
- HTML5
- CSS3 (Grid Layout, Flexbox, Custom Properties)
- JavaScript ES6 puro (State Machine para encadeamento de operações)
